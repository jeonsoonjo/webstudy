정규표현식

1. regular expression
2. 특정 문자의 매칭 결과를 판단하기 위한 표현식이다
3. 자바스크립트의 정규식
	var regExp=/정규식/;
4. 정규식 메타 문자
	1) a		: 문자 a를 의미
	2) a|b		: 문자 a or b
	3) .		: 모든 문자를 의미(1글자)
	4) ^a		: 문자 a로 시작
	5) a$		: 문자 a로 끝
	6) a*		: 문자 a가 여러 번 나온다. 안 나와도 된다
	7) a+		: 문자 a가 여러 번 나온다. 최소 1번은 나온다
	8) a{3}		: 문자 a가 3번 반복해서 나온다.
	9) a{3,}	: 문자 a가 3번 이상 나온다
   10) a{3,5}	: 문자 a가 3번 이상 그리고 5번 이하 나온다.
5. 정규식 문자클래스
	1) [a]		: 문자 a를 의미
	2) [.]		: 문자 .를 의미
	3) [ab]		: 문자 a or b
	4) [^ab]	: 문자 a 그리고 문자 b를 제외
	5) ^[ab]	: 문자 a or 문자 b로 시작
	6) [ab]$	: 문자 a or 문자 b로 끝
	7) [a-z]	: 문자 a~z 중 하나 (소문자를 의미)
	8) [A-Z]	: 문자 A~Z 중 하나 (대문자를 의미)
	9) [0-9]	: 숫자 0~9 중 하나
   10) [가-힣]	: 한글 가~힣 중 하나
   11) \.		: 문자 .을 의미
   12) \s		: 공백문자(space)를 의미
   13) \S		: 공백문자가 아니다(not space)를 의미 -> 대문자는 not을 의미함
   14) \d		: 숫자(digit)를 의미(0~9를 의미, 위 9번과 같음)
   15) \D		: 숫자가 아니다(not digit)를 의미
   16) \w		: 대문자/소문자(word)를 의미
   17) \W		: 대문자/소문자가 아니다(not word)를 의미
   
   
연습
1. 문자 j를 1번 이상 포함하고 있는가?
	var reg = /j+/;
2. 소문자를 1번 이상 포함하고 있는가?
	var reg = /[a-z]+/;
3. 소문자이거나 대문자인가?
	var reg = /[a-z] | [A-Z]/;
    var reg = /[a-zA-Z]/; -> 공백 없이 쭉 이어서 쓰면 된다
4. 숫자가 3자리인가?
	var reg = /[0-9]{3}/;
	var reg = /\d{3}/;
5. 소문자가 아닌가?
	var reg = \[^a-z]/;
	

실전
1. 휴대전화번호가 맞는가? (모두 010으로 시작한다)
	var regPhone = /^010-[0-9]{3,4}-[0-9]{4}$/; -> ^010 시작한다 의미, - 하이픈, [0-9]{3,4} 가운데 숫자가 3이상 4이하 자리 의미, [0-9]{4}$ 마지막 4자리 끝
	var regPhone = /^010-\d{3,4}-\d{4}$/;
2. 아이디가 맞는가?
	1) 5~20자이다 -> a{5,20}
	2) 소문자, 숫자, 특수문자(_, -) 2가지만 사용한다 -> [a-z][0-9][_-]
	3) 소문자나 숫자로 시작한다 -> ^[a-z]|[0-9] -> ^[a-z0-9] 같이 적어주면 된다
	4) 위 3가지 조건을 만족해야 한다
	var regID = /^[a-z0-9][a-z0-9_-]{4,19}$/; -> 앞에 소문자나 숫자로 한 글자 시작했기 때문에 4~19 자리가 필요하다 
3. 이메일이 맞는가?
	1) id@domain.com
	2) id@domain.co.kr
	var regEmail = /^[a-z0-9][a-z0-9_-]*@[a-zA-Z0-9]+([.][a-zA-Z]{2,}){1,2}$/;
4. 비밀번호가 맞는가?
	1) 8~16자이다
	2) 대문자, 소문자, 숫자, 모든 특수문자를 사용할 수 있다
	3) 모든 1개 이상은 포함되어야 한다
		(1) 대문자 포함되었는지 검사		?=.*[A-Z]
		(2) 소문자 포함되었는지 검사		?=.*[a-z]
		(3) 숫자가 포함되었는지 검사		?=.*[0-9]
		(4) 특수문자가 포함되었는지 검사	?=.*[`~!@#$%^&*()-_=+\|{[}\];:'"<,>.?/] -> ] 닫는 걸로 인식할 수도 있으니 앞에 \를 붙인다
	var regPW = /^[A-Za-z0-9`~!@#$%^&*]{8,16}$/;
